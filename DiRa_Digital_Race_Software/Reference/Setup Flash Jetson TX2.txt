SETUP FLASH JETSON TX2

Install Nvidia SDK Manager
https://developer.nvidia.com/embedded/downloads

Download and run install file

Open SDK Manger launcher using Ubuntu Launcher or Open it from terminal with command 

$ sdkmanager

Login and then install the corresponding required package for Jetson TX2
This will required upto 19GB of free space

Connect host computer to Jetson TX2 through USB-Micro cable and flash the TX2
